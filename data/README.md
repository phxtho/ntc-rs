# Data

## Colour Map

colour map is sourced from [name-that-color](https://github.com/pbojinov/name-that-color);

### Structure

`colour = ["002E20", "Burnham", 0, 46, 32, 114, 255, 23]`

```
{
    0: hex,
    1: name,
    2: red,
    3: green,
    4: blue,
    5: hue,
    6: saturation,
    7: luminosity
}
```
